Bem-vindo a página de administração da SkillShare

Utilize o menu na barra no topo para acessar aos diversos conteúdos

Criar nova questão para o banco de questões

Para criar uma nova questão no banco de questões, escolha um curso no qual a questão irá pertencer, em seguida preencher os campos e escolha qual será a alternativa correta

Postagem de um curso

Preencha todos os campos abaixo para realizar o upload de um novo curso

Registro de atividades da plataforma

Nesta seção você pode visualizar todas as atividades dos usuários do sistema e saber as modificações, quem modificou e quando modificou algo na plataforma

Criar prova final para um curso

Selecione um curso para ver questões relacionadas com ele e adicione questões na lista de selecionadas
para elaborar uma prova final para o curso. Caso o curso já tenha uma prova final, a antiga será
substituida pela nova

Listagem de cursos

Aqui estão todos os cursos da plataforma. Selecione um para realizar alterações ou excluí-lo.

Adicionar novo conteúdo à biblioteca

A biblioteca da SkillShare é um local onde ficam postados arquivos PDFs no qual os alunos tem acesso a este
material, dessa forma podem incrementar ainda mais seus conhecimentos

Gerir biblioteca

Escolha um material para editar suas informações, visualizar o conteúdo ou excluir

Gerir questões do banco de questões

Para visualizar as questões de um curso, editá-la ou excluí-la, selecione o respectivo curso abaixo para exibir as questões vínculadas a ele

Cadastro de usuários

Escolha um tipo de usuário para visualizar seus
campos de informações e realizar o cadastro

Cadastrar usuário

Preencha os campos para cadastrar o usuário

Gerir usuários

Escolha um tipo de usuário para visualizar seus dados e gerir aqueles
que estão em nosso banco de dados

Criar nova pílula de conhecimento

A biblioteca da SkillShare é um local onde ficam postados arquivos PDFs no qual os alunos tem acesso a este
material, dessa forma podem incrementar ainda mais seus conhecimentos

Gerir pílulas de conhecimento

Escolha um curso para visualizar todas as pílulas referentes a ele, podendo excluí-las ou editar suas informações

Inserir nova notícia/evento

Notícias e eventos são conteúdos que ficam disponíveis aos alunos em suas páginas principais,
com o intuíto de que os gestores e administradores possam fornecer conhecimentos diversos que não se
relacionam necessariamente com uma disciplina e mante-los atualizados com os acontecimentos do mundo

Gerir notícias e eventos

Escolha entre notícias e eventos para visualizar os registros do respectivo tipo, podedendo
ver os dados de cada um para editar ou excluir

Cadastre uma nova turma para que seus cursos possuam alunos!
Toda turma é atrelada a um curso, possui alunos e um tutor, que ajudará no desenvolvimento dos seus alunos.
